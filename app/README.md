# CloneGuard 🛡️
A decentralized APK authenticity verification system built on Solana blockchain. Developers register their app's SHA256 hash + Ed25519 signature on-chain, and users can instantly verify whether any APK is genuine or a clone — without any central authority.

**Tech Stack:** Rust + Anchor · Solana Devnet · React + Vite · Phantom Wallet · SHA256 · Ed25519
